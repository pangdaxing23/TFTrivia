TFTrivia React Native App
=========================

Install Instructions
--------------------
1. `yarn`
2. `react-native run-ios`

   or

   `react-native run-android` // (must have device connected)


* Button turns blue when questions are loaded.



Test Instructions
-----------------
1. `yarn test` or `yarn test -- --watch`

------------------------------------------------------------------------------

`Enjoy. ⛺️🔥`
